Expressions and Statments

A fragment of code that produces a value is called an expression.

Every value that is written literally (such as 10 or "television") is an expression.

A program is simply a list of statments

The simplest kind of statment is an expression with a semicolon after it. Ex:

1;

!false;


Variables

In order for your program to remember things, it needs to be placed and held into a value. Holding these values is called a "variable."

Ex: var caught = 5 * 5;

Adding "var" indicates that we are writing out a variable, the rest is just, what are we naming the varible. The name can be anything we want it to be.

Ex:

var ten = 10;

console.log(ten * ten);

// -> 100


Just because you've named a variable, doesn't mean that varible's name is the end all be all. The "=" operator can be used to disconnect the name of the variable value and have it pointed to another one.

Ex:

var mood = "light";

console.log(mood);

// -> light

mood = "dark";

console.log(mood);

// -> dark
