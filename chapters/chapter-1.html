Numbers

Simple enough. They, for the most part appear as numbers.

Special numbers = Infinity, positive and negative. NAN = Not a number


Strings

Strings are represented via text. They are always written in enclosing quotes:
Quotes must have a start and an end when describing text

String cannot be divided, multipled, or subtracted. The "+" operator can still be used on it however


Unary Operators

Not all operators are symbols. Some are written as words

Ex: Console.log(typeof 4.5)

// -> number

console.log(typeof "x")

// -> String

Operators that use two values are called binary operators, while those that take one are called unary operators

Ex: Console.log(-(10-2))

// -> -8


Boolean Values

When you need a value that can distinguish between two possibilities like true and false, you'll need boolean values

Ex: console.log(3 > 2) This can also apply o strings

// -> true

console.log (3 < 2)

// -> false

Ex: Console.log ("Aardvark" < "Zoroaster")

// -> true

Strings are ordered more or less alphabeticly. Uppercase letters are always "less" than lowercase ones

Comparisons

console.log(3 > 2)

// -> true

console.log(3 < 2)

// -> false
